---
title: "Document Status Check"
description: "Check the status of the document uploaded via /create/submission. "
api: "GET /document/{documentId}/checkStatus"
---

This endpoint retrieves the current status of a document identified by `documentId`. The `documentId` must be provided as a path parameter, immediately following `/document/` and preceding `/checkStatus`.

<ParamField path="documentId" type="string" required={true}>
  Enter `documentId` as a path parameter.
</ParamField>

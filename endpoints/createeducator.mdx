---
title: "Create User"
description: "Create and add a new user in the system. "
api: "POST /user"
authMethod: "bearer"
---

This API call creates a new educator user on the platform. You must specify the user's name, email, role, and status.

### Request Parameters

<ParamField body="firstName" type="string" required>
  Enter the name for the new user
</ParamField>

<ParamField body="lastName" type="string" required>
  Enter the last name for the new user
</ParamField>

<ParamField body="role" type="string" required>
  Assign `educator` or `student` as the role for the new user.
</ParamField>

<ParamField body="email" type="string" required>
  Enter the email for the new educator.
</ParamField>

<ParamField body="status" type="number" required="false">
  Enter the status for the new user Use `1` for active status and `0` for inactive status.
</ParamField>

<ParamField body="courses" type="string[]" required>
  Specify the subjects for the new user using the corresponding subject IDs.
</ParamField>

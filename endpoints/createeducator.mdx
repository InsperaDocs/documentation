---
title: "Create Educator"
description: "Create a new educator use via API."
api: "POST /user"
---

This API call creates a new educator user on the platform. You must specify the user's name, email, role, and status.

<ParamField body="firstName" type="string" required>
  Enter the name for the new educator.
</ParamField>

<ParamField body="lastName" type="string" required>
  Enter the last name for the new educator.
</ParamField>

<ParamField body="role" type="string" required>
  Assign `educator` as the role for the new user.
</ParamField>

<ParamField body="email" type="string" required>
  Enter the email for the new educator. 
</ParamField>

<ParamField path="param" type="string">
  Enter the status for the new eduactor. Use `1` for active status and `0` for inactive status. 
</ParamField>

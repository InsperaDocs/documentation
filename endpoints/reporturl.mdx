---
title: "Report URL"
description: "Retrieve the URL for an Originality Report."
api: "GET /document/{documentId}/mode/{mode}"
---

This endpoint initiates the Originality Report API by passing the necessary path parameters, such as `documentId` and `mode`. In return, the API provides a URL that can be used to display the originality report.

### Request Parameters

<ParamField path="documentId" type="string" required>
  Provide the unique identifier for the document. 
</ParamField>

<ParamField path="mode" type="string" required>
  Specify the view mode for the document's Originality Report, either `view` or `edit`.
</ParamField>

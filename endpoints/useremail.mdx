---
title: "User Info (Email)"
description: "Retrieve user information via email."
api: "GET /user-info"
---

This **GET** request allows you to retrieve user information based on their email address. By passing the user's email as a query parameter, the endpoint returns detailed data associated with that specific user.

### Request Parameters

<ParamField query="email" type="string" required="false">
  Enter the user's email.

  <Warning>
    Ensure the email is associated with an `active status` user in the system.
  </Warning>
</ParamField>

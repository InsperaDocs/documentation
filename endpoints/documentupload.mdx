---
title: "Document Upload"
description: "Upload a document via API"
api: "POST /create/submission"
---

This POST request initiates the document similarity check. The endpoint acts as the entry point for the system to process and analyze the submitted document. The response will include a `message`, `documentId`, and `presignedS3Url` where the document should be uploaded.

### Request Parameters

When making a request, only four parameters are required, and the rest are optional to include as needed.

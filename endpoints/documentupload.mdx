---
title: "Create Submission "
description: "Upload a document via API"
api: "POST /create/submission"
authMethod: "bearer"
---

This POST request initiates the document similarity check. The endpoint acts as the entry point for the system to process and analyze the submitted document. The response will include a `message`, `documentId`, and `presignedS3Url` where the document should be uploaded.

### Request Parameters

When making a request, only four parameters are required, and the rest are optional to include as needed.

#### Required Parameters

<ParamField body="documentTitle" type="string" required>
  The title of the document that will be uploaded.
</ParamField>

<ParamField body="author" type="string" required>
  The name of the document's author.
</ParamField>

<ParamField body="email" type="string" required>
  The email of the document's author.
</ParamField>

<ParamField body="docType" type="string" required>
  Document type that is being uploaded.

  <Expandable title="Supported Types">
    Document type that is being uploaded. Supported types:

    - `application/pdf` or `.pdf` (PDF files)
    - `text/plain` or `.txt` (Plain text files)
    - `text/html` or `.html` (HTML files)
    - `application/xml` or `.xml` (XML files)
    - `application/msword`, `application/doc`, `application/ms-doc`, or `.doc`
    - `application/vnd.openxmlformats-officedocument.wordprocessingml.document`, `word`, `docx`, or `.docx`
    - `application/vnd.oasis.opendocument.text` or `.odt`
    - `application/rtf` or `.rtf`
    - `application/wordperfect` or `.wpd`
    - `application/postscript` or `.ps`
    - `application/vnd.apple.pages` or `.pages`
    - `application/vnd.ms-powerpoint`, `application/mspowerpoint`, `application/powerpoint`, `application/x-mspowerpoint`, or `.ppt`
    - `application/vnd.openxmlformats-officedocument.presentationml.presentation` or `.pptx`
    - `application/vnd.openxmlformats-officedocument.presentationml.slideshow` or `.ppsx`
    - `application/vnd.oasis.opendocument.presentation` or `.odp`
    - `application/vnd.apple.keynote` or `.key`
    - `application/vnd.ms-excel`, `application/excel`, `application/x-excel`, `application/x-msexcel`, or `.xls` (Excel 97-2003 files)
    - `application/vnd.openxmlformats-officedocument.spreadsheetml.sheet` or `.xlsx`
    - `application/vnd.oasis.opendocument.spreadsheet` or `.ods`
    - `application/vnd.apple.numbers` or `.numbers`
    - `application/json` or `.json`
    - `text/csv` or `.csv`
    - `.md`
  </Expandable>
</ParamField>

#### Optional Parameters

<ParamField body="translatedLanguage" type="string[]">
  Languages specified for the Translated Langauge Similarity analysis, up to 3 maximum.

  <Warning>
    Make sure to specify the languages via code. E.g., for English, input "en"
  </Warning>
  <Expandable title="Supported Languages">
    - `Albanian (sq)`, `Bosnian (bs)`, `Bulgarian (bg)`
    - `Croatian (hr)`, `Czech (cs)`, `Danish (da)`
    - `Dutch (nl)`, `English (en)`, `Estonian (et)`
    - `Finnish (fi)`, `French (fr)`, `German (de)`
    - `Greek (el)`, `Hungarian (hu)`, `Italian (it)`
    - `Latvian (lv)`, `Lithuanian (lt)`, `Macedonian (mk)`
    - `Norwegian (no)`, `Polish (pl)`, `Portuguese (pt)`
    - `Romanian (ro)`, `Russian (ru)`, `Serbian (sr)`
    - `Slovak (sk)`, `Slovenian (sl)`, `Spanish (es)`
    - `Swedish (sv)`, `Turkish (tr)`
  </Expandable>
</ParamField>

<ParamField body="assignmentId" type="string">
  The corresponding assignmentId where the document will be uploaded.
</ParamField>

<ParamField body="institutionName" type="string" required="false">
  The name of the institution where the document will be uploaded.
</ParamField>

<ParamField body="enableAIDetection" type="boolean">
  Enable to check for content potentially generated by AI.
</ParamField>

<ParamField body="enableReferenceDetection" type="boolean">
  Toggles reference detection (optional, enabled by default).
</ParamField>

<ParamField body="reSubmissions" type="number">
  Sets how many times resubmission is allowed.
</ParamField>

<ParamField body="subjectId" type="string">
  The corresponding subjectId where the document will be uploaded.
</ParamField>

<ParamField body="maxLimit" type="number">
  The maximum number of words allowed in a submission.
</ParamField>

<ParamField body="archive" type="boolean">
  Indicates whether the submission should be archived for collusion detection.
</ParamField>

<ParamField body="enableContextSimilarity" type="boolean">
  Enable to check for potential contextual similarity.
</ParamField>

<ParamField body="sentenceThresholds" type="object">
  Define the percentage at which a sentence is considered contextually similar.

  `"contextualSimilaritiesThreshold": 65`

  ```json Example icon=webhook
  "sentenceThresholds": {
  "contextualSimilaritiesThreshold": 65 // specify as number
  }
  ```

  <Warning>
    We recommend using the default contextual similarity threshold for optimal results.
  </Warning>
</ParamField>

<ParamField body="sources" type="object">
  Define sources to include or exclude form the document originality check.

  ```json Example icon=webhook
  "sources": { // object
    "includeSources": [ //array
      "https://example.com/source1", 
      "https://example.org/source2"
    ],
    "excludeSources": [ //array
      "https://exclude.com/sourceA",
      "https://exclude.org/sourceB"
    ]
  }
  ```
</ParamField>

<ParamField body="sourcesThreshold" type="number">
  Specify the percentage threshold for excluding sources from the Originality   Report.For example, setting this value to 5% means that any sources contributing less than 5% similarity will be automatically excluded from the report.
</ParamField>
